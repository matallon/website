Mary Tallontire is a London based digital artist working to create projects through code. Currently working towards a BSc in Digital Arts Computing at Goldsmiths University, London. 

Pieces currently revolve around worlding, procedural generation, simulations and story worlds. Presently working with the Unity game engine as well as openFrameworks with C++, in the future they hope to delve more into digital storytelling, immersive experiences, and game development to create more expansive worlds and stories with more interaction. 

They're always looking into new technologies and softwares, utlising coding and problem solving skills with these to create new projects and experiences.

Projects 

Terra Nullis 

3D Procedural generation piece created using the Unity game engine. Generates new landscape data using Perlin noise on each iteration of the program, and fills this with new variations of objects and humans. As the program runs it sporadically generates a new terrain and plethora of objects to fill this topography, creating brand new scenery each time it regenerates.

After

A 2D pixel art interactive virtual world of a post apocalyptic future where climate change has destroyed our current civilization.

Created in p5.js and all assets made in Procreate and Photoshop. 

Illustration work 

Freelance illustration work done under the name ‘Murkybucket,’ consisting of animated and still line drawings sold as commissions on my Etsy account. All illustrations are created using the Procreate app or using Photoshop. Commissioned pieces have been used at album covers, tattoos and one was used by singer Sigrid to promote a competition. 

Other smaller pieces - mostly made in openFrameworks and c++
